RuangTV
PANDUAN SETUP WINDOWS

Setup RuangTV
di Windows

Ikuti langkah-langkah berikut untuk menjalankan backend RuangTV di laptop/PC Windows kamu. Estimasi waktu: 10–15 menit.

1
Install Node.js
WAJIB

Node.js adalah runtime JavaScript yang dibutuhkan untuk menjalankan server RuangTV. Download versi LTS (Long-Term Support).

Download
🌐 https://nodejs.org
→ Klik tombol "LTS" → Download → Install (next next finish)
Verifikasi instalasi — buka CMD dan ketik:
node --version
→ Harus muncul: v20.x.x atau v22.x.x
Cara buka CMD: Tekan Windows+R → ketik cmd → Enter
2
Atur Struktur Folder yang Benar
WAJIB

Berdasarkan screenshot kamu, semua file masih flat di satu folder. File auth.js, database.js, routes.js, dan websocket.js harus masuk ke subfolder src\.

📁 RuangTV-1\ ← folder utama kamu
  📄 server.js
  📄 package.json ← buat baru (lihat step 3)
  📄 .env ← buat baru (lihat step 4)
  📄 README.md
  📁 src\ ← BUAT FOLDER INI
    📄 auth.js ← pindahkan ke sini
    📄 database.js ← pindahkan ke sini
    📄 routes.js ← pindahkan ke sini
    📄 websocket.js ← pindahkan ke sini
  📁 uploads\ ← buat folder kosong
  📁 public\ ← taruh file HTML di sini
    📄 ruangtv-dashboard.html
    📄 ruangtv-schedule.html
    📄 ruangtv-upload.html
    📄 ruangtv-player.html
  📁 node_modules\ ← otomatis dibuat saat npm install

Cara cepat pakai CMD:

Buka CMD di folder RuangTV-1, lalu ketik:
mkdir src
mkdir uploads
mkdir public
move auth.js src\
move database.js src\
move routes.js src\
move websocket.js src\
REM Pindahkan HTML ke public\
copy ruangtv-*.html public\
Cara buka CMD di folder: Di File Explorer, klik address bar → ketik cmd → Enter. CMD langsung terbuka di folder itu.
3
Buat File package.json
WAJIB

Buat file baru bernama package.json di folder utama RuangTV-1. Bukan di dalam src\, tapi di root folder.

Isi package.json (copy paste ke Notepad, simpan sebagai package.json)
{ "name": "ruangtv-backend", "version": "1.0.0", "description": "RuangTV Digital Signage Backend", "main": "server.js", "scripts": { "start": "node server.js", "dev": "node --watch server.js" }, "dependencies": { "cors": "^2.8.5", "dotenv": "^16.3.1", "express": "^4.18.2", "multer": "^1.4.5-lts.1", "sql.js": "^1.10.2", "ws": "^8.14.2" } }
Cara buat file dengan Notepad: Buka Notepad → paste isinya → Save As → nama file: package.json → Save as type: All Files → simpan di folder RuangTV-1
4
Buat File .env (Konfigurasi)
WAJIB

File .env berisi konfigurasi server. Buat di folder utama RuangTV-1 (sama level dengan server.js).

Isi file .env
PORT=3001 JWT_SECRET=ruangtv-secret-ruangprint-2025 ADMIN_USERNAME=admin ADMIN_PASSWORD=ruangprint2025
PORT
Port server (default 3001, jangan diubah kecuali konflik)
JWT_SECRET
Kunci enkripsi token. Bisa diganti dengan string apapun
ADMIN_USERNAME
Username login dashboard HO
ADMIN_PASSWORD
Password login dashboard HO
⚠️ Nama filenya harus persis .env (ada titik di depan, tidak ada ekstensi). Di Notepad → Save As → nama: .env → Save as type: All Files
5
Install Dependencies (npm install)
WAJIB

Perintah ini akan mengunduh semua library yang dibutuhkan server dan membuat folder node_modules. Butuh koneksi internet.

Buka CMD di folder RuangTV-1, lalu ketik:
npm install
Proses ini 1–3 menit tergantung koneksi internet.
Setelah selesai akan muncul folder node_modules dan file package-lock.json.
Jika ada error, coba ini:
npm install --legacy-peer-deps
6
Jalankan Server
FINISH

Setelah semua langkah di atas selesai, jalankan server dengan perintah berikut:

Opsi 1 — Double-click file:
📄 START-RuangTV.bat ← double-click langsung
Opsi 2 — Via CMD:
node server.js
Jika server berhasil jalan, kamu akan melihat:
╔══════════════════════════════════════════╗ ║ RuangTV Backend Server ║ ╠══════════════════════════════════════════╣ ║ HTTP → http://localhost:3001 ║ ║ WS → ws://localhost:3001/ws ║ ╚══════════════════════════════════════════╝ 📺 TV Keys (pasang di Smart TV tiap cabang): DMB: 04c4ab27ec783b37... CTR: 8868d1ade23ccc32... ...

Buka browser dan akses:

Dashboard HO
http://localhost:3001
API Health
http://localhost:3001/health
Login
admin / ruangprint2025
7
Setup Smart TV di Setiap Cabang
SMART TV

Setelah server jalan di laptop HO, Smart TV di setiap cabang perlu diarahkan ke server. Cara paling mudah: buka browser di Smart TV dan akses URL Player.

URL Player per cabang (ganti IP_LAPTOP_HO dengan IP Windows kamu)
# Cari IP laptop: buka CMD → ketik ipconfig → lihat IPv4 http://192.168.1.xxx:3001/ruangtv-player.html?branch=DMB http://192.168.1.xxx:3001/ruangtv-player.html?branch=CTR http://192.168.1.xxx:3001/ruangtv-player.html?branch=GDS http://192.168.1.xxx:3001/ruangtv-player.html?branch=CGK http://192.168.1.xxx:3001/ruangtv-player.html?branch=BKS
Syarat: Laptop HO dan Smart TV cabang harus terhubung ke internet yang sama (atau VPN), atau laptop HO harus punya IP publik.

Untuk cabang berbeda kota: Deploy server ke VPS (cloud server) seperti Niagahoster, DigitalOcean, atau Railway.app — bisa dilihat di step berikutnya.
Cari IP laptop kamu:
ipconfig
→ Cari baris "IPv4 Address" → itu IP kamu
8
Deploy ke VPS (Untuk Cabang Berbeda Kota)
OPSIONAL

Kalau cabang ada di kota berbeda, server harus di-deploy ke VPS supaya bisa diakses dari mana saja. Rekomendasi yang paling mudah dan murah:

Railway.app
Gratis ($5 credit/bulan) — paling mudah, deploy pakai GitHub
Niagahoster VPS
~Rp 50-100rb/bulan — lokal Indonesia, latency rendah
DigitalOcean Droplet
$6/bulan — stabil, banyak tutorial
Deploy ke VPS Linux (setelah punya server):
# 1. Upload file ke server scp -r RuangTV-1/ user@IP_SERVER:/home/user/ # 2. Install Node.js di server curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - sudo apt-get install -y nodejs # 3. Install & jalankan cd RuangTV-1 && npm install node server.js # 4. Pakai PM2 (auto-restart) npm install -g pm2 pm2 start server.js --name ruangtv pm2 save && pm2 startup
Butuh Bantuan Lebih Lanjut?
Kalau ada error saat setup, screenshot pesan errornya dan tanyakan ke Claude.
Step berikutnya: Sambungkan Frontend ke API dan Update TV Player real-time.
